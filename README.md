In this project is presented the creation of the Meal Lab App. 
More specifically, in the first part of the project the data from the source REST APIs by the source https://www.themealdb.com/api.php is deserialized, so it can be use in the second part of the project.
The second part is consists to the GUI application development, via JavaFX.
To sum up, in the Meal Lab App, the end user has the opportunity to explore new recipes and create custom lists.
